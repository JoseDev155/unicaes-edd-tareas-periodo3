## Ejericio 8: Gestor de Inventario con Exportación de Datos

**Descripción:**<br />
Sistema donde los productos se agrupan por tipo, marca y modelo. El usuario puede
exportar el inventario completo o parcial a CSV, HTML o XML.

**Temas cubiertos:**<br />
* Listas multinivel (tipo → marca → modelo).
* Métodos para guardar información en diferentes formatos.
* Interfaz amigable con HTML/CSS tipo tabla con filtros.

## Estructura del proyecto

* `arbol-de-decision/`: Contiene la imagen del árbol de decisión.
* `documento/`: Contiene el documento entregable del Parcial 3.
* `capturas/`: Contiene las capturas de pantalla de la ejecución del programa.
* `css/`, `js` e `index.html`: Contienen los archivos para la interfaz web del sistema.
